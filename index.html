About this dashboard

The Berkeley Energy Dashboard provides public access to building-level utility data for the UC Berkeley campus — electricity, natural gas, water, and steam/chilled water.

Data source

Utility meter data is collected by the campus metering network and managed in SkySpark, the university's energy analytics platform. Building footprints are provided by UC Berkeley Facility Services GIS. (Preview mode currently shows representative sample data while the live connection is finalized.)

Solar production

Several campus buildings generate on-site solar power. For these buildings, meters record net consumption (after solar). The dashboard can display both net and gross (pre-solar) values where production metering is available.

Update frequency

Data refreshes daily. Each day's values are finalized the following morning.

Questions or feedback
